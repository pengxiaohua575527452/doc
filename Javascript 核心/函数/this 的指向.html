• 函数 this 值的指向

  • 如果使用function 关键字 声明一个函数 function fn(){}

    • 在严格模式下
      • 作为对象的方法调用
        • this 指向调用的对象
      • 作为函数调用
        • this 指向 undefined
      • 备注：
        • 采用严格模式的方式
          • "use strict"
          • class
    • 在非严格模式下
      • 作为对象的方法调用
        • this 指向调用的对象
      • 作为函数调用
        • this 指向 window

  • 如果使用 箭头函数 const fn = () => {}

    • 箭头函数中的this 值指向 所在环境上下文的 this

  • 全局作用域环境下的this 指向 window
