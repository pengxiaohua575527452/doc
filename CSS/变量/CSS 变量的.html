• CSS 变量声明和使用

    • 变量的声明
    • :root{ --var-name: value;}
    • :root

      通过选择器 定义变量的作用域范围

      只有当前选择器的 子元素 和自身 能够使用定义的变量

      子选择器定义的变量 会覆盖 祖先选择器定义的变量 值

      :root 表示全局作用域

    • --var-name

      变量的名称

      --开头是规范

      -作为连接符

    • value

      变量的值

    • 变量的使用
    • .child{ color: var(--color);}
    • var()

      使用标量的函数

    • --color

      变量的名称

    • 在控制台 css 值依然显示 var(--color)

      不会 显示计算的结果
